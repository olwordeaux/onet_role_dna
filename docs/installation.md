# Installation

## Stable release

To install onet_role_dna, run this command in your terminal:

```sh
uv add onet_role_dna
```

Or if you prefer to use `pip`:

```sh
pip install onet_role_dna
```

## From source

The source files for onet_role_dna can be downloaded from the [Github repo](https://github.com/olwordeaux/onet_role_dna).

You can either clone the public repository:

```sh
git clone https://github.com/olwordeaux/onet_role_dna
```

Or download the [tarball](https://github.com/olwordeaux/onet_role_dna/tarball/main):

```sh
curl -OJL https://github.com/olwordeaux/onet_role_dna/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd onet_role_dna
uv sync
```
